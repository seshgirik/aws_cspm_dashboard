üõ°Ô∏è Implemented Security Features

Stateless authentication using JSON Web Tokens with 24-hour expiration

BCrypt Password Encoding

Passwords encrypted with BCrypt (strength 12) before storage

Role-Based Access Control

USER and ADMIN roles with hierarchical permissions

Method-Level Security

@PreAuthorize annotations for fine-grained access control

CSRF Protection

Cross-Site Request Forgery protection (disabled for APIs, enabled for forms)

CORS Configuration

Configured to allow specific origins with credentials

Security Headers

CSP, X-Frame-Options, XSS-Protection, HSTS headers configured

Session Management

Stateless sessions with maximum 1 concurrent session per user

Input Validation

Jakarta Validation for request DTOs with custom constraints

Exception Handling

Custom authentication entry point for unauthorized access

H2 Console Security

Frame options configured for development database access

Health and metrics endpoints secured with admin-only access
