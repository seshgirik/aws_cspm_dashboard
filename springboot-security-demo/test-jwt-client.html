üîê JWT Client Test - How to Send JWT Tokens

Step 1: Login to Get JWT Token

cURL Command: curl -X POST http://localhost:8080/auth/login \ -H "Content-Type: application/json" \ -d '{"username":"admin","password":"admin123"}' JavaScript Code: fetch('http://localhost:8080/auth/login', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ username: 'admin', password: 'admin123' }) }) .then(response => response.json()) .then(data => { localStorage.setItem('token', data.token); })

Step 2: Stored JWT Token

How Token is Stored: // Browser LocalStorage localStorage.setItem('token', 'eyJhbGciOiJIUzI1NiJ9...'); // Browser SessionStorage sessionStorage.setItem('token', 'eyJhbGciOiJIUzI1NiJ9...'); // Variable in memory const token = 'eyJhbGciOiJIUzI1NiJ9...';

Step 3: Send Requests with JWT Token

Authorization: Bearer <your-jwt-token> Content-Type: application/json
cURL Command with Token: TOKEN="eyJhbGciOiJIUzI1NiJ9..." curl -X GET http://localhost:8080/api/user/profile \ -H "Authorization: Bearer $TOKEN" JavaScript Code with Token: const token = localStorage.getItem('token'); fetch('http://localhost:8080/api/user/profile', { method: 'GET', headers: { 'Authorization': `Bearer ${token}` // ‚Üê JWT Token Here! } }) .then(response => response.json()) .then(data => console.log(data))
